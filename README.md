# 🌍 Global AI Adoption in Education Dashboard (2015–2026)

## 📊 Project Overview
This project analyzes global AI adoption trends in education using real-world dataset.

## 🚀 Features
- Time series analysis of AI adoption
- Regional comparison
- Urban vs Rural insights
- Gender gap analysis
- Interactive dashboard

## 🛠 Tools Used
- Python
- Pandas
- Chart.js
- HTML/CSS
- AI-assisted analysis
